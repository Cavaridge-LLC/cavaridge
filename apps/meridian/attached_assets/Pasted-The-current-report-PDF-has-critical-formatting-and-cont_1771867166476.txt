━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

New report structure (each section starts on new page where indicated):

PAGE 1 — COVER
  CONFIDENTIAL watermark
  MERIDIAN logo
  "IT Due Diligence Report"
  Target company name (large)
  Industry badge, Deal code
  "Prepared for: {org_name}"
  "Assessment Date: {date}"
  "Platform: MERIDIAN v2.0 — Cavaridge, LLC"

PAGE 2 — TABLE OF CONTENTS (auto-generated with page numbers)

PAGES 3-4 — EXECUTIVE SUMMARY [page-break before]
  - Investment Verdict (with score gauge visual)
  - Target Profile
  - Key Risk Findings (5-7 bullets)
  - Evidence Confidence Warning (if applicable)
  - Cost & Timeline Snapshot
  - Top 3 Conditions Before Close
  - Severity distribution bar + evidence coverage matrix

PAGE 5 — RISK SCORE OVERVIEW [page-break before]
  - Composite score gauge (large)
  - Pillar score horizontal bar chart (6 bars)
  - Evidence confidence matrix
  - Scoring methodology note (2 sentences)

PAGES 6-11 — PILLAR ASSESSMENTS (1-2 pages each) [page-break before each]
  For each of 6 pillars:
  - Pillar name + score + confidence badge
  - AI-written assessment narrative
  - Key consolidated findings table
  - Strengths / Concerns bullets
  - Remediation priorities with costs

PAGES 12-15 — CONSOLIDATED FINDINGS REGISTER [page-break before]
  - Summary table (severity × count × theme)
  - Critical findings: full detail callout boxes
  - High findings: full detail callout boxes  
  - Medium findings: consolidated table
  - Low findings: simple list

PAGES 16-17 — TECHNOLOGY STACK [page-break before]
  - Stack items grouped by category in a clean table
  - EOL items highlighted in red
  - Column: Category | Item | Version | Status | Confidence

PAGE 18 — BASELINE ALIGNMENT [page-break before]
  - If baseline exists: gap table with severity colors
  - If not: "Baseline comparison pending — configure acquirer profile 
    in MERIDIAN Settings to enable gap analysis."

PAGES 19-20 — INTEGRATION ROADMAP [page-break before]
  - Timeline visual
  - Phase table: Phase | Timeline | Tasks | Critical | Est. Cost
  - Cost summary table: Tier | Hours | Rate | Subtotal
  - Risk factors callout boxes

PAGE 21 — NETWORK TOPOLOGY [page-break before]
  - Visual diagram (HTML/CSS)
  - Node and connection summary table

PAGE 22 — RECOMMENDATIONS & NEXT STEPS [page-break before]
  - AI-generated prioritized action items
  - 30/60/90 day priorities
  - Estimated total integration budget range

APPENDICES [page-break before each]:
  A: Full Findings Register (all original findings, not consolidated)
  B: Document Inventory table
  C: Methodology & Glossary
  D: Disclaimer
